# Menghitung Luas Permukaan Tabung

## Rumus
Luas = 2πr(r+t) atau Atap + Alas + Selimut  
Atap = π × r²  
Alas = π × r²  
Selimut = 2 × π × r × t  

## Input
- r = jari-jari alas tabung
- t = tinggi tabung
- π = 3.14159

## Pseudocode

```
  BEGIN
    INPUT r
    INPUT t
    SET atap = π * r * r
    SET alas = π * r * r
    SET selimut = 2 * π * r * t
    SET luas = atap + alas + selimut
    OUTPUT "Luas permukaan tabung = ", luas
  END
```
